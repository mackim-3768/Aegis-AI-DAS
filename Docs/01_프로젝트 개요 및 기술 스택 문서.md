# Project Brief: Aegis-AI : DAS (Driver Assistance System)

## 1. Project Overview
**Aegis-AI : DAS**는 Google의 **FunctionGemma** 모델을 활용한 지능형 차량 주행 보조 시스템(Advanced Driver Assistance System)의 작동 원리를 시각화하고 시뮬레이션하는 안드로이드 애플리케이션입니다. 

본 앱은 50가지의 차량 센서 데이터(Context Tools)와 제어 장치(Action Tools)가 AI 에이전트의 추론에 의해 어떻게 유기적으로 작동하는지 사용자에게 직관적인 '디지털 트윈' 환경으로 제공하는 것을 목표로 합니다.

## 2. Core Value & Vision
- **Explainable AI (XAI):** AI가 왜 특정 주행 제어를 수행했는지(Thought Process)를 투명하게 시각화합니다.
- **Interactive Simulation:** 고정된 시나리오가 아닌, 사용자가 직접 데이터를 조작하며 AI의 반응을 실시간으로 확인합니다.
- **Next-Gen UI/UX:** 미래지향적이고 정갈한 디자인을 통해 복잡한 차량 데이터를 효율적으로 전달합니다.

## 3. Tech Stack
| Category | Technology |
| :--- | :--- |
| **OS/Platform** | Android (Min SDK 26+) |
| **Language** | Kotlin |
| **UI Framework** | Jetpack Compose |
| **Architecture** | MVVM (Model-View-ViewModel) |
| **State Management** | Kotlin Coroutines & StateFlow |
| **Image Loading** | Coil |
| **Animation** | Lottie, Compose Animation API |
| **DI (Optional)** | Hilt (for scalability) |

## 4. Design Concept: "Pristine Lab"
- **Theme:** Ultra-Clean White & Light Gray (#F8F9FA)
- **Style:** Glassmorphism (Semi-transparent with Blur), Minimalist
- **Point Color:** Electric Blue (#007AFF) for active states, Teal (#00A896) for safe states
- **Visual Goal:** 애플(Apple)의 깔끔한 감성과 테슬라(Tesla)의 기술적 정교함이 결합된 실험실 환경 구현

## 5. Main Feature Architecture (4-Tab Structure)
1. **Tab 1: Aegis Digital Twin**
   - 차량의 상단 뷰(Top-view) 및 실내 뷰(Interior-view) 레이아웃.
   - Action Tool 실행 시 실시간 애니메이션 오버레이 표시.
2. **Tab 2: Neural Grid (Data Manager)**
   - 50개 도구(25 Context + 25 Action)의 상태를 카드 형태로 나열.
   - 디버그 모드 활성 시 각 센서 값 실시간 편집 기능.
3. **Tab 3: Thinking Log (Reasoning Stream)**
   - FunctionGemma의 `thought` 필드 및 JSON 로그 실시간 스트리밍.
4. **Tab 4: System Control (Scenario Injector)**
   - CPU/GPU 연산 가속 설정.
   - 사전 정의된 엣지 케이스 시나리오 실행기.

## 6. Development Rules for AI
- 모든 UI는 **Jetpack Compose**로 작성할 것.
- 50개의 도구 상태는 단일 진실 공급원(Single Source of Truth)인 **ViewModel의 StateFlow**를 통해 관리할 것.
- 가독성을 위해 각 탭별로 Composable 함수를 별도 파일로 분리할 것.
- 화이트 테마를 기반으로 하며, 그림자(Shadow)와 블러(Blur)를 활용하여 레이어 간 입체감을 줄 것.
