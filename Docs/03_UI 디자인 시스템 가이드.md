# Design System Guide: Aegis-AI : DAS (Pristine Lab Theme)

## 1. Color Palette (Semantic Tokens)

모든 색상은 눈의 피로도를 낮추는 **Off-White** 베이스와 신뢰감을 주는 **Electric Blue** 액센트를 중심으로 구성합니다.

| Token | Hex Code | Usage |
| --- | --- | --- |
| **Background** | `#F8F9FA` | 전체 앱의 메인 배경색 (Pure White 대비 눈부심 방지) |
| **Surface (Card)** | `#FFFFFF` | 카드, 버튼 등 요소의 기본 배경색 |
| **Primary (Accent)** | `#007AFF` | 활성화된 상태, 강조 텍스트, 진행률 표시 (Apple Blue) |
| **Secondary (Safe)** | `#00A896` | 안전 상태, 긍정적 피드백 (Clean Teal) |
| **Warning** | `#FFCC00` | 주의가 필요한 센서 데이터 (Amber) |
| **Danger** | `#FF3B30` | 긴급 제어, 위험 상황 발생 (System Red) |
| **Text Primary** | `#1C1C1E` | 메인 타이틀 및 본문 텍스트 |
| **Text Secondary** | `#8E8E93` | 캡션, 비활성 텍스트, 단위(Unit) 표시 |

## 2. Shape & Elevation (Geometry)

깔끔하고 현대적인 느낌을 위해 넓은 곡률과 얇은 경계선을 사용합니다.

* **Corner Radius:**
* `Card & Large Button`: **24.dp** (둥글고 부드러운 인상)
* `Small Chip / Input Field`: **12.dp**


* **Border (Stroke):**
* `Weight`: **0.5.dp** (매우 얇고 섬세한 라인)
* `Color`: `Text Secondary` 컬러에 `alpha 0.2f` 적용


* **Shadow (Elevation):**
* 기본 상태: `elevation(2.dp)` - 아주 미세한 그림자
* 활성/강조 상태: `elevation(8.dp)` - 부드럽게 떠오르는 느낌



## 3. Glassmorphism Spec

화이트 테마의 깊이감을 더하기 위해 반투명 효과를 적극 활용합니다.

* **Opacity (Alpha):** `0.8f` (배경이 비치는 투명도)
* **Blur Effect:** `Modifier.blur(16.dp)` (카드 뒷배경 흐림 처리)
* **Layering:** 하얀 배경 위에 반투명 카드를 쌓아 레이어 간 공간감을 형성합니다.

## 4. Animation Policy

모든 상태 변화는 갑작스럽지 않게, 유기적으로 연결되어야 합니다.

* **Standard Duration:** `500ms` (너무 빠르지도, 느리지도 않은 우아한 속도)
* **Easing:** `FastOutSlowInEasing` (자연스러운 가속과 감속)
* **Key Transitions:**
* **Value Change:** 수치 변경 시 숫자가 부드럽게 롤링되거나 Fade 전환.
* **Card Expansion:** 디버그 모드 진입 시 카드가 커지는 효과에 `spring` 물리 엔진 적용 고려.
* **Action Trigger:** 액션 발생 시 해당 아이콘이 미세하게 맥동(Pulse)하는 효과.

## 5. Kotlin Implementation (Sample for AI)

AI 비서가 `Theme.kt`와 `Color.kt`를 작성할 때 참고할 참조 코드입니다.

```kotlin
// Color.kt 정의 예시
val AegisBackground = Color(0xFFF8F9FA)
val AegisPrimary = Color(0xFF007AFF)
val AegisGlassWhite = Color(0xFFFFFFFF).copy(alpha = 0.8f)

// Shape.kt 정의 예시
val AegisShapes = Shapes(
    medium = RoundedCornerShape(24.dp),
    small = RoundedCornerShape(12.dp)
)

// AnimationSpec 정의 예시
val AegisDefaultAnimation = tween<Float>(
    durationMillis = 500,
    easing = FastOutSlowInEasing
)

```
